# Drawing Application

This project is a simple drawing application that allows users to create artwork using various tools.

## Features

- **Brush Selection**: Users can choose from different types of brushes for drawing.
- **Color Change**: Ability to change the color of the brush and background.
- **Erase**: Capability to erase specific parts of the drawing.
- **Clear Canvas**: Option to clear the entire drawing canvas.
- **Canvas**: Utilizes a canvas element for drawing.
